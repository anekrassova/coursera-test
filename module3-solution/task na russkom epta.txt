Для выполнения этого задания следуй шагам:

1. **Создай аккаунт и репозиторий на GitHub**:
   Если еще не сделал, зарегистрируйся на GitHub и создай репозиторий для курса. Этот репозиторий будет местом, где ты будешь хранить решения заданий.

2. **Настрой GitHub Pages**:
   Просмотри видео из модуля 1 (начало), в котором объясняется, как настроить репозиторий, чтобы публиковать веб-страницы через GitHub Pages. Это позволит просматривать свои страницы через домен вида `username.github.io`.

3. **Создай папку для задания**:
   В репозитории создай новую папку для этого задания, например, `module3-solution`. Внутри этой папки создай файл `index.html`, который будет твоей главной страницей.

4. **Создай один файл HTML**:
   Вся реализация будет в одном HTML-файле. В нем должно быть несколько секций, как показано в макетах для разных устройств: настольный компьютер, планшет, и мобильный телефон.

5. **Добавь файл CSS**:
   В папке с решением создай папку `css` и помести туда файл стилей, например, `style.css`. Не используй встроенные стили в HTML (внутри атрибутов `style`), всё должно быть в CSS-файле.

6. **Используй Bootstrap**:
   В проекте нужно максимально использовать Bootstrap для верстки. Скачай начальные файлы Bootstrap (например, из лекции 25, часть 2), и помести их в папку с решением.

7. **Создай адаптивную верстку**:
   Макеты:
   - Для настольного компьютера используй классы `col-md-*` (пример: `col-md-12` для секций, занимающих всю ширину экрана).
   - Для планшета используй классы `col-sm-*`.
   - Для мобильных устройств — классы `col-xs-*`.

8. **Навбар (navbar)**:
   Добавь навигационную панель (navbar) с названием компании "Food, LLC", выровненным по левому краю. Для настольного и планшетного вида не должно быть никаких других кнопок. Для мобильного вида добавь кнопку с тремя горизонтальными полосками, которая при нажатии будет открывать выпадающее меню.

9. **Меню для мобильных устройств**:
   Добавь в выпадающее меню три элемента: Chicken, Beef, Sushi. Меню должно занимать всю ширину экрана и иметь отличающийся цвет фона.

10. **Заголовок страницы**:
   Центрируй заголовок "Our Menu" с помощью класса Bootstrap для выравнивания текста.

11. **Добавь высокую секцию**:
   Создай одну большую секцию, которая будет занимать всю ширину экрана. Эта секция должна быть настолько высокой, чтобы вызывать необходимость прокрутки. Установи высоту в 1000px или наполни ее большим количеством текста.
